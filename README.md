# Smart Expense Tracker

## Description
This is a simple expense tracker project made using Python and Streamlit.  
It helps to record daily expenses and shows basic analysis like total spending and category-wise expenses.


## Features
- Add daily expenses
- View total spending
- Category-wise analysis
- Daily expense graph
- Monthly prediction
- Alerts for high spending


## Tools Used
- Python
- Pandas
- Streamlit
- Matplotlib
- Scikit-learn


## Files
- app.py
- expenses.csv
- requirements.txt
- README.md


## How to Run
1. Install required libraries:
pip install -r requirements.txt

2. Run the project:
streamlit run app.py
