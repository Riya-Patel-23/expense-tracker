# Smart Expense Tracker

## Overview
This project is a simple expense tracking application built using Python and Streamlit.  
It helps in recording daily expenses and gives a basic analysis of spending patterns.  
It also includes a simple prediction feature to estimate future expenses.

## Features
- Add daily expenses
- View total spending
- Category-wise expense analysis
- Daily spending trend
- Pie chart for expense distribution
- Monthly expense prediction
- Alerts for high spending categories

## Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib
- Scikit-learn

expense-tracker/

## How to Run

1. Clone the repository:
git clone https://github.com/your-username/expense-tracker.git

2. Open the project folder:
cd expense-tracker

3. Install required libraries:
pip install -r requirements.txt

4. Run the application:
streamlit run app.py

## Usage
- Enter expense details from the sidebar
- View charts and insights on the dashboard
- Check predicted expense for next month
